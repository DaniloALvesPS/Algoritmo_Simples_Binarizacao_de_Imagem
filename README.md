# Algoritmo_Simples_Binarizacao_de_Imagem
 implementação em Python para transformar uma imagem colorida para níveis de cinza (0 a 255) e para binarizada (0 e 255), preto e branco.
 
Binarização de imagem é um processo de transformação de uma imagem em tons de cinza ou colorido em uma imagem em preto e branco, onde cada pixel da imagem resultante é classificado como preto ou branco (ou 0 e 1, respectivamente).

Na binarização, um valor limite (ou threshold) é escolhido, e cada pixel na imagem é comparado com esse valor limite. Se o valor do pixel é maior que o valor limite, ele é convertido para branco. Se o valor do pixel é menor ou igual ao valor limite, ele é convertido para preto.

O objetivo da binarização é simplificar a imagem, atendendo a quantidade de informações necessárias para representá-la. Isso pode ajudar em vários tipos de processamento de imagem, como detecção de bordas, reconhecimento de objetos, segmentação de imagens, entre outros.

Existem diferentes métodos de binarização de imagens, cada um adequado a diferentes situações. Alguns exemplos incluem a binarização global, onde um único valor limite é aplicado em toda a imagem, e a binarização adaptativa, onde um valor limite é ajustado para cada região da imagem, levando em conta as diferenças locais de iluminação e contraste.
